# StudioZero

## Development Setup

1.  Create and activate a virtual environment:
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    ```
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Code Quality

This project uses `black` for formatting and `pyright` for type checking.

-   **Format code**: `black .`
-   **Check types**: `pyright`
